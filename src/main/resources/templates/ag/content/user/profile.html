Using this platform, you can buy or sell items, share items and book reservation for community resources.
Actions

To give access to the platform, you need to add a user for each unit in your community. First, you need to prepare a file containing a list of users. Each line in the file should contain unit identification and email address of the owner separated by a comma. Then upload the file. Email with instructions as to how to login will be sent to the users. Upload users

To enable reservation, add facilities for which reservation feature would be helpful for community users. Add Facility
